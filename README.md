# mern-walkthrough

To Run: (Outdated)
  cd mern/server
  node --env-file=config.env server

  ( Ctrl + C to stop)

  Open new terminal

  cd mern/client
  npm run dev

To Run: (Updated) (Using Concurrently)
  cd mern
  npm run dev

  // Concurrently will automatically start the server and client side from
  // the root directory. Config is in /mern/package.json
